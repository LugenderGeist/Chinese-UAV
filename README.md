# Упрощение полиномов, описывающих аэродинамику БВС

Для проекта по разработке системы управления БВС констркуции Tailsitter появилась необходимость проверить СУ на другом БВС, описание которого найдено в одном изисточник литературы.

Написан код который упрощает полиномы, описывающие БВС, что необходимо для упрощения вывода уравнений обратной кинематики.

Статьи:
1. Yunjie Yang. Active Disturbance Rejection Control of a Flying-Wing Tailsitter Hover Flight / Yunjie Yang // DOI 10.1109/IROS.2018.8594470
2. Yunjie Yang. Robust proportional incremental nonlinear dynamic inversion control of a flying-wing tailsitter / Yunjie Yang // DOI 10.1177/0954410020926657
